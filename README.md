# TextMint

Simple and fast text cleaning tools that run entirely in your browser.  
No uploads. No tracking. No login.

🔗 Live site: https://textmint.pages.dev/

---

## Features

- Remove duplicate lines
- Remove empty lines
- Trim whitespace
- Remove duplicate words
- JSON to Excel converter

All tools process data locally in your browser for speed and privacy.

---

## Why TextMint?

- Lightweight and fast
- No external dependencies
- Privacy focused (no uploads)
- Simple and clean UI
- Mobile friendly

---

## Tools

### Remove Duplicate Lines
Remove repeated lines from text instantly.

### Remove Empty Lines
Delete blank lines while preserving formatting.

### Trim Whitespace
Remove leading and trailing spaces from each line.

### Remove Duplicate Words
Remove repeated words while preserving order.

### JSON to Excel
Convert JSON arrays into CSV format for Excel.
